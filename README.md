# cmdshortcut 快捷命令

Note: This program currently only supports **Simplified Chinese**.

本程序目前仅支持简体中文.

Used to set a shortcut key to run a program.

用于设置运行一个程序的快捷键.

After running the program, follow the prompts to continue.

运行程序后，按照提示继续.

Press `Alt Shift A` to view detailed information, including commonly used shortcuts and **the directory of configuration files**.

按 `Alt+Shift+A` 查看详细信息，包含常用快捷键和**配置文件所在目录**.

`CustomKey1` represents the program executed after pressing `Alt 1`, `CustomKey2` represents the program executed after pressing `Alt 2`, and so on.

`CustomKey1` 代表按下 `Alt+1` 后执行的程序，`CustomKey2` 代表按下 `Alt+2` 后执行的程序，以此类推.

After the program runs, it **will be hidden in the taskbar tray**.

程序运行后会隐藏在**任务栏托盘**中.

You can also close the program by pressing `Alt Shift F`.

同时也可以按` Alt+Shift+F` 关闭程序.

The default password for the administrator panel is in the configuration file.

管理员面板的默认密码在配置文件中.
